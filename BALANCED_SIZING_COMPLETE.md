## Changes Applied:

### 1. **Balanced Typography Scale**
- **Base font size**: 2rem (optimized for classroom viewing)
- **H1**: 3rem (was 6rem)
- **H2**: 2.5rem (was 5rem)
- **H3**: 2.25rem (was 4rem)
- **H4**: 2rem (same as body)
- **H5**: 1.75rem
- **Body text**: 2rem
- **Secondary text**: 1.75rem

### 2. **Component-Specific Adjustments**

#### **Headings**
- Section titles: 1.25x base size (25px at default)
- Transparent background boxes for all headings
- Uppercase styling with letter spacing for differentiation
- Inline-block display for fitted backgrounds

#### **Vocabulary**
- Word display: 1.3x-1.5x base size (26-30px)
- Part of speech & pronunciation: Same as content (20px)
- Vietnamese translation: 1.2x-1.3x base size
- Differentiated by border thickness and background opacity

#### **Exercises**
- Exercise titles: 1.15x base size
- Exercise content: Same as body text
- Answer sections: Same as body text
- Differentiated by background colors

#### **Content**
- All regular content: Base size (20px default)
- Dialogues: Same as body text
- Inline vocabulary: Same as body text
- Tables and lists: Inherit parent size

### 3. **Visual Differentiation (Not Size)**
- **Headings**: Transparent glass backgrounds, uppercase text
- **Vocabulary**: Thicker borders, gradient overlays, hover effects
- **Exercises**: Colored backgrounds, expandable sections
- **Regular content**: Standard presentation

### 4. **Icon Adjustments**
- Navigation icons: 1.5x base size (reduced from 2.5x)
- Audio buttons: 1.2x base size
- UI controls: Proportionally sized

## Result:
The app now maintains consistent, balanced font sizes across all content types, with visual styling (not size) used to create hierarchy and differentiation. This ensures excellent readability for classroom projection while maintaining clear content organization.
