## Keyboard Shortcuts:
- **Ctrl/Cmd + Plus**: Increase font size
- **Ctrl/Cmd + Minus**: Decrease font size
- **Arrow Keys**: Navigate between sections (structured mode)
- **Space**: Speak vocabulary (when focused)

## Tips for Classroom Use:
1. Press **F11** for fullscreen presentation
2. Use font size **24-28px** for large classrooms
3. Toggle Vietnamese translations on/off as needed
4. Use "One by One" mode for vocabulary drills
