# Vocabulary Format Examples

## Multi-line Format
```
bustling (adj)
/ˈbʌslɪŋ/
hối hả, nhộn nhịp, náo nhiệt
carry out
/ˈkæri aʊt/
tiến hành (nghiên cứu, nhiệm vụ)
```

## Tab-separated Format
```
bustling (adj)	hối hả, nhộn nhịp, náo nhiệt
carry out	/ˈkæri aʊt/	tiến hành (nghiên cứu, nhiệm vụ)
come down with	bị ốm (vì bệnh gì)
```

## Comma-separated Format
```
bustling (adj), hối hả, nhộn nhịp, náo nhiệt
carry out, /ˈkæri aʊt/, tiến hành (nghiên cứu, nhiệm vụ)
come down with, bị ốm (vì bệnh gì)
```
