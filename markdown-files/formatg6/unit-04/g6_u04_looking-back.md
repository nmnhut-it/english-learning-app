**Điền hình thức so sánh hơn của tính từ**   
**So sánh hơn:**   
tính từ ngắn: có một vần   
tính từ dài: hai vần trở lên  
**Công thức so sánh hơn:**   
tính từ ngắn \+ đuôi \-er  \+ than …  
more \+ tính từ dài \+ than ….  
than: so với   
1\. fast (nhanh) → faster (nhanh hơn)   
2\. beautiful → more beautiful  
3\. noisy → noisier  

4. expensive → more expensive   
5. hot → hotter   
6. exciting → more exciting   
7. quiet → more quiet/quieter   
8. heavy → heavier   
9. large → larger
