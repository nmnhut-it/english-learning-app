# **So sánh hơn và so sánh nhất trong tiếng Anh**

## **Tính từ ngắn (1 âm tiết)**

| Dạng thường | So sánh hơn | So sánh nhất |
| :---- | :---- | :---- |
| tall | taller | tallest |
| smart | smarter |   |
| big | bigger | biggest |
| hot | hotter | hottest |
| sad | sadder | saddest |

**Công thức**:

- So sánh hơn: tính từ \+ **er**  
- So sánh nhất: the \+ tính từ \+ **est**

**Lưu ý đặc biệt**:

- Tính từ kết thúc bằng **e**: thêm **r/st** (nice → nicer → nicest)  
- Tính từ có 1 âm tiết kết thúc bằng 1 phụ âm, trước đó là 1 nguyên âm: gấp đôi phụ âm cuối (big → bigger → biggest)  
- Tính từ kết thúc bằng **y**, trước đó là phụ âm: đổi **y** thành **i** (happy → happier → happiest)

## Tính từ dài (2 âm tiết trở lên)

| Dạng thường | So sánh hơn | So sánh nhất |
| :---- | :---- | :---- |
| beautiful | more beautiful | most beautiful |
| difficult | more difficult | most difficult |
| expensive | more expensive | most expensive |
| interesting | more interesting | most interesting |
| comfortable | more comfortable | most comfortable |

**Công thức**:

- So sánh hơn: **more** \+ tính từ  
- So sánh nhất: **the most** \+ tính từ

## Một số tính từ bất quy tắc

| Dạng thường | So sánh hơn | So sánh nhất |
| :---- | :---- | :---- |
| good | better | best |
| bad | worse | worst |
| far | farther/further | farthest/furthest |
| little | less | least |
| many/much | more | most |
